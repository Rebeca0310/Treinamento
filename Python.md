# **Python**### ***Tipos de dados compostos***## Além dos tipos básicos de Python (int, float, str e bool), temos tipos complexos, que podem contém mais de uma valor. Os principais em python são:* Listas de elementos: **lits*** Tuplas: **tuple*** Conjuntos: **set*** Dicionarios: **dict**#### **Lista*** Uma lista em Python pode conter zero ou mais elementos de qualquer tipo! A lista armazena os valores numa ordem, começando com o índice 0. Para uma lista, basta definir os elementos dentro de colchetes.* O acesso ao elemento de uma lista se dá pelo seu índice, ou seja, a posição do elemento dento da lista! **Lembrando que o 1° elemento está na posição 0 (zero).*** Uma característica muito legal e python é a  **possibilidade de acesso o último elemento** sem saber quantos elementos temos na lista! É só passar uma posição negativa: **começando pelo -1(menos um).*** Uma outra característica é que podemos ter repetições de elementos, sem problema nenhum.* Para adicionar elementos numa lista, podemos utilizar o **método** **append()*** Para remover elementos, é um pouquinho diferente (inclusive de outras linguagens), então, muita atenção: Utilizamos o **comando del** passando a **posição do elemento a ser removido.*** Temos outra opção para remover, utilizando o método **remove()**. Para isso, temos que **passar o elemento eu queremos remover**, e não a posição.* Para altera um valor, basta atribuir o novo valor na posição que se quer substituir.* Para saber a quantidade de elementos de uma lista utilizamos a **função len().*** No python as strings se comportam como listas. Temos como acessar os caracteres de uma string pela posição como se uma lista fosse#### **Tuplas*** Tuplas são muito parecidas com listas! A grande diferença é que, **uma vez criada a tupla, ela não pode mais mudar!!** Ou seja, nada de acrescentar novos elemento, remover ou alterar elementos. Podemos criar tuplas com parênteses ao invés de colchetes* Outra forma de criar uma tupla basta atribuir valores separados por vírgulas: o Python cria a tupla e armazena todos os valores na ordem repassada* O acesso ao elemento de uma tupla é igual na lista, pelo índice, ou seja, a posição do elemento dentro da lista! **Lembrando que o 1° elemento está na posição 0 (zero),** as posições sempre tem que estar **entre colchetes**#### **Conjuntos** * Um conjunto em python se assemelha muito ao conceito matemático. Os elementos **não tem uma ordem definida e também não podem ser repetidas**! Para criar um conjunto, basta usar chaves na criação dos elementos. * Se a gente tentar criar um conjunto com elementos repetidos, **não dá erro**, mas o elemento não é adicionado em repetição! E a ordem dos dados não se mantém* Para incluir elementos num conjunto, basta utilizar o **método add()*** Buscar elementos num conjunto, já que eles estão em ordem , é um tanto quanto "diferente", por dois motivos   * A ordem em que o elemento é retornado só os deuses do Python sabe  * Não tem uma forma direta de buscar um elemento sem removê-lo do conjunto* Para buscar o elemento, basta usar o **método pop(),**ele retorna o elemento **e o remove do conjunto**#### **Dicionários*** Em python os dicionários possuem uma chave que remete a um valor. **Chave** é o valor que estamos procurando e o **Valor** seria o resultado mapeado daquela chave * Um elemento de um dicionários sempre será uma junção de **chave + valor** * Os dicionários, portanto, é diferente das estruturas que vimos ate agora* Ao invés de receber um elemento, ele armazena PARES de chave e valor* As chaves devem ser únicas e casa uma possui um valor* Para criar um dicionário, basta passar os pares de chave-valor entre chaves {}* As chaves dos dicionários são únicas: isso nos dá uma vantagem e facilidade para encontrar os valões relacionados. Basta passar o valo da chave entre colchetes e o dicionário retorna pra você rapidinho* Nas outras linguagens essas estruturas são conhecidas como hashsets ou hashmaps. O importante é você saber que o acesso aos valores se dá em tempo recorde * 3 métodos importantes do dicionários são Keys(), values() e items()  * o Keys() retorna todas as chaves do dicionário   * O Values() retorna todos os valores do dicionário.   * O items() retorna todos os pares de chave: valor do dicionário * Para alterar um valor mapeado por um chave, parece com a atualização dos valores de uma lista: só atribuir o novo valor á chave correta!#### #### **Conversões**Para cada uma das estruturas compostas que vimos nessa aula,temos uma função no python para transformar um tipo de estrutura no outro (de modo bem similar as funções dos tipos básicos , o chamado casting)As funções são: list(), tuple(), set() e dict()Desse modo, podemos transformar uma estrutura na outra (menos o dict que vai ficar sozinho nesse, porque ele precisa de pares:chaves-valor)
