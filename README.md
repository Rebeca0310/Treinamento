# 🎲 Jogo do Número Secreto

Um jogo simples e divertido em JavaScript onde o usuário deve adivinhar o número secreto entre 1 e 10. O projeto utiliza HTML, CSS e JavaScript para criar uma interface interativa e responsiva.

## 🚀 Funcionalidades
- Geração de número secreto aleatório sem repetição até esgotar o limite.
- Feedback imediato ao usuário:
  - "O número secreto é maior" ou "O número secreto é menor".
  - Mensagem de vitória com número de tentativas.
- Botão de reinício para jogar novamente.
- Integração com **ResponsiveVoice** para leitura em voz alta das mensagens.
- Interface estilizada com gradiente e responsividade para diferentes tamanhos de tela.

## 🛠️ Tecnologias utilizadas
- **HTML5** para estrutura da página.
- **CSS3** para estilização e responsividade.
- **JavaScript (ES6)** para lógica do jogo.
- **ResponsiveVoice API** para síntese de voz.
